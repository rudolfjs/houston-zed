## 0.1.0 - 2026-04-30
### Added
* Initial structure
* Initial changelog and release-note tracking
* Initial theme by [Astro](https://vscodethemes.com/e/astro-build.houston/houston) ported to Zed
* Tag-driven release automation, changie enforcement, and Zed extension metadata/docs alignment

* Require changie fragments for deletions, guard release tag moves against stale remote refs, tighten release tag validation, and skip bot-triggered release reruns
* Prevent tag releases from failing when extension.toml already contains the release version
